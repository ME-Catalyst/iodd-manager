# IODD Manager Roadmap

## Version 1.0 (Current - November 2025)
### Completed Features
- ✅ IODD XML parsing and validation
- ✅ Device import from ZIP/IODD files
- ✅ Nested ZIP package support (1-level deep)
- ✅ Multi-file bulk import
- ✅ SQLite database storage
- ✅ REST API with FastAPI
- ✅ React-based web interface
- ✅ Device management (CRUD operations)
- ✅ Interactive Menus tab with full IODD variable support
- ✅ Standard IO-Link variable parsing
- ✅ Parameter visualization and export
- ✅ Asset management and image gallery
- ✅ XML source viewer
