## 5-Minute Setup

### 1. View Current Theme
- Navigate to Admin Console
- Click "Appearance" tab
- See your currently active theme

### 2. Try a Preset
- Browse the 4 built-in themes:
  - **Greenstack**: Default dark theme
  - **Forest Green**: Deep forest tones
  - **Midnight Green**: Dark teal accents
  - **Light Green**: Clean light theme
- Click "Activate" on any theme
- Theme applies instantly!

### 3. Create Custom Theme
- Click "Create Custom Theme"
- Choose a base preset
- Customize colors using the editor
- Click "Preview" to see live changes
- Click "Save" when satisfied

### 4. Edit Colors
Theme editor has 7 color categories:
- Primary (buttons, links)
- Secondary (supporting elements)
- Accent (highlights)
- Semantic (success, warning, error)
- Surfaces (backgrounds)
- Borders (dividers)
- Text (typography)

Each color has:
- Visual picker
- Hex input
- Live preview
- Contrast checker (text colors)

### 5. Export/Import
- **Export**: Click download icon on theme card
- **Import**: Click "Import" button, select JSON file

## Key Features

### Brand Protection
- Brand green (#3DB60F) is locked
- Cannot be changed in any theme
- Ensures brand consistency

### Live Preview
- See changes in real-time
- Preview before saving
- Easy rollback

### Accessibility
- WCAG AA/AAA checking
- Contrast ratio calculator
- Accessibility warnings

## Common Tasks

### Switch to Light Mode
1. Go to Appearance tab
2. Find "Light Green" theme
3. Click "Activate"

### Make Text Brighter
1. Edit your custom theme
2. Go to "Text" tab
3. Adjust "Foreground" color
4. Check contrast ratio
5. Save changes

### Duplicate a Theme
1. Export the theme you like
2. Import the JSON file
3. Edit and customize
4. Save with new name

## Tips

- Start with a preset that's close to your goal
- Use preview mode extensively
- Check contrast for readability
- Name themes descriptively
- Export backups of custom themes

## Limitations

- Brand green (#3DB60F) cannot be changed
- One active theme at a time
- Custom themes are stored locally

## Quick Fixes

**Theme not applying?**
- Refresh the page
- Check browser console
- Verify API is running

**Can't save theme?**
- Check all required colors are set
- Verify brand green is #3DB60F
- Check network connection

**Low contrast warning?**
- Increase color difference
- Use WCAG checker suggestions
- Test with actual content
